# Stone V2

Stone V2 is a new governed repo for a premium RTL-native Shopify theme platform built for Israeli Shopify developers and agencies.

## Repo shape

- `HARNESS/` shared governance, templates, and checks
- `docs/architecture/` canonical product and architecture inputs
- `project/` Shopify theme runtime root
- `state/` canonical governed project state files
- `workspace/` plans, intake, build, verify, handoff, QA, notes, and logs

## Important rule

The five files under `docs/architecture/` are the initial source of truth for Codex planning in this repo.
