# VERIFY REPORT Template

## Scope

- `Verified`: State the bounded unit and changed surfaces under review.

## Verdict

- `Verified`: PASS, FAIL, or BLOCKED.

## Checks Run

- `Verified`: List the exact commands, manual checks, or file validations performed.

## Findings

- `Verified`: Record concrete failures or state that no findings were observed.

## Evidence

- `Verified`: Reference files, logs, diffs, or command output.

## Follow-Up

- `Inference`: Record the next bounded action if work remains.
