# Phase 3 Merchandising Foundation Boundary Risks v1

1. `Inference`: Merchandising can sprawl immediately if the shared product-card contract is not kept restrained.
2. `Inference`: A first broader slice could quietly smuggle in commerce actions, recommendation logic, or template coupling if stop conditions are weak.
3. `Verified`: The mitigation is to limit the slice to one snippet, one section, one template, base CSS, locale updates, read-only collection/product access, and exact forbidden behaviors.
